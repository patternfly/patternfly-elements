Removes `-Height` and `-Width` CSS custom properties in favour of `size`.

```css
pfe-progress-indicator {
  /* --pfe-progress-indicator--Height: 2rem; */
  /* --pfe-progress-indicator--Width: 2rem; */
  --pfe-progress-indicator-size: 2rem;
}
```
