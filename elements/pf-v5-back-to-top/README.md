# Back to top

The **back to top** component is a shortcut that allows users to quickly navigate to the top of a lengthy content page.


##  Installation
Load `<pf-v5-back-to-top>` via CDN:

```html
<script src="https://jspm.dev/@patternfly/elements/pf-v5-back-to-top/pf-v5-back-to-top.js"></script>
```

Or, if you are using [NPM](https://npm.im), install it

```bash
npm install @patternfly/elements
```

Then once installed, import it to your application:

```js
import '@patternfly/elements/pf-v5-back-to-top/pf-v5-back-to-top.js';
```

## Usage

```html
<pf-v5-back-to-top href="#top">Back to Top</pf-v5-back-to-top>

```
