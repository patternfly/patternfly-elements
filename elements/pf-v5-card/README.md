# PatternFly Elements Card
         
Read more about Card in the [PatternFly Elements Card documentation](https://patternflyelements.org/components/card)

##  Installation

Load `<pf-v5-card>` via CDN:

```html
<script src="https://jspm.dev/@patternfly/elements/pf-v5-card/pf-v5-card.js"></script>
```

Or, if you are using [NPM](https://npm.im), install it

```bash
npm install @patternfly/elements
```

Then once installed, import it to your application:

```js
import '@patternfly/elements/pf-v5-card/pf-v5-card.js';
```

## Usage

```html
<pf-v5-card>
  <h2 slot="header">Card header</h2>
  <p>This is the pf-v5-card body.</p>
  <a href="#" slot="footer">Footer link</a>
</pf-v5-card>
```
